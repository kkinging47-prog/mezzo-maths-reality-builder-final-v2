# Mezzo Maths Reality Builder

A clean deployable React/Vite app for practical mathematics missions where students use maths to build real-world objects.

## Local setup

```bash
npm install
npm run dev
```

Open the local Vite URL, usually `http://localhost:5173`.

## Production build

```bash
npm run build
npm run preview
```

## Vercel deployment

- Framework: Vite
- Install command: `npm install`
- Build command: `npm run build`
- Output directory: `dist`

The included `vercel.json` supports browser refresh on nested routes.
